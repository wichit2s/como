ลำดับความสำคัญของ
==================================================

+-------+--------------+----------------------------------+
| Level | Operators    | Process Sequence / คำอธิบายประกอบ |
+=======+==============+==================================+
|   1   | ::           |  ซ้ายไปขวา                        |
+-------+--------------+----------------------------------+
|       | ++  --       |                                  |
+-------+--------------+----------------------------------+
|   2   | ()           |   ซ้ายไปขวา                       |
+-------+--------------+----------------------------------+
|       | []           |                                  |
+-------+--------------+----------------------------------+
|       | . ->         |                                  |
+-------+--------------+----------------------------------+
|       | ++  --       |                                  |
+-------+--------------+----------------------------------+
|       | ~  !         |                                  |
+       +--------------+----------------------------------+
|       | +  -         |                                  |
+       +--------------+----------------------------------+
|   3   | &  *         |  ขวาไปซ้าย                        |
+       +--------------+----------------------------------+
|       | new  delete  |                                  |
+       +--------------+----------------------------------+
|       | sizeof       |                                  |
+       +--------------+----------------------------------+
|       | (type)       |  เปลี่ยนชนิดข้อมูล                    |
+-------+--------------+----------------------------------+
|   4   | .* ->*       |  เข้าถึงสมาชิก                      |
+-------+--------------+----------------------------------+
|   5   | *  /  %      |                                  |
+-------+--------------+----------------------------------+
|   6   | +  -         |                                  |
+-------+--------------+----------------------------------+
|   7   | <<  >>       |                                  |
+-------+--------------+----------------------------------+
|   8   | <  >  <=  >= |                                  |
+-------+--------------+----------------------------------+
|   9   | ==   !=      |                                  |
+-------+--------------+----------------------------------+
|  10   |  &           |                                  |
+-------+--------------+----------------------------------+
|  11   |  ^           |                                  |
+-------+--------------+----------------------------------+
|  12   |              |  bitwise OR                      |
+-------+--------------+----------------------------------+
|  13   |  &&          |                                  |
+-------+--------------+----------------------------------+
|  14   |              |  relation OR                     |
+-------+--------------+----------------------------------+
|  15   |              |  กำหนดค่า และ กำหนดค่าแบบผสม       |
+-------+--------------+----------------------------------+
|  16   |    ,         |                                  |
+-------+--------------+----------------------------------+
